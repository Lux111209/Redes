¿Qué son los componentes activos en la Red?

Los componentes activos en una red son aquellos que requieren energía eléctrica para funcionar. Estos dispositivos son responsables de gestionar el flujo de datos a través de la red, dirigir las señales a sus destinos correctos, y en algunos casos, amplificar o regenerar las señales para mantener la integridad de los datos.

Descripción técnica de cada componente activo
  • Switch: Es un dispositivo de red que conecta múltiples dispositivos dentro de una misma red y utiliza direcciones MAC para enviar los datos solo al puerto correspondiente, mejorando la eficiencia de la red.
  • Router: Dispositivo que se encarga de dirigir el tráfico de datos entre diferentes redes, seleccionando la mejor ruta para que los datos lleguen a su destino. Es fundamental para conectar una red local a Internet.
  • Tarjeta de red: También conocida como NIC (Network Interface Card), es un componente que permite a un dispositivo conectarse a una red. Puede ser una tarjeta interna o un adaptador externo con puertos Ethernet o WiFi.
  • Interruptor inteligente Wifi: Dispositivo que controla otros dispositivos a través de una conexión WiFi. Permite encender o apagar dispositivos de manera remota mediante una aplicación o un asistente de voz.
  • Access Point: Es un dispositivo que permite a los dispositivos inalámbricos conectarse a una red cableada mediante WiFi. Los puntos de acceso se utilizan para ampliar el alcance de una red inalámbrica.
Herramientas utilizadas para realizar cableado estructurado

  • Ponchadora:
    • Descripción técnica: Herramienta utilizada para insertar los cables en los conectores, como los módulos RJ45, asegurando que los contactos se perforen adecuadamente y se establezca una conexión eléctrica segura.
  • Crimpadora:
    • Descripción técnica: Se utiliza para crimpar o engarzar conectores RJ45 o RJ11 al final de un cable de red (UTP o STP). Esta herramienta prensa los contactos metálicos dentro del conector, asegurando que hagan buen contacto con los hilos de cobre del cable.
  • Desforradora:
    • Descripción técnica: Esta herramienta se utiliza para retirar la cubierta externa del cable UTP sin dañar los conductores internos. Permite acceder a los pares trenzados para prepararlos para la crimpadora o la ponchadora.
  • Tester/probador de cables:
    • Descripción técnica: Dispositivo utilizado para verificar la conectividad y la correcta disposición de los hilos en un cable de red. Detecta problemas como cortes, cortocircuitos o pares mal cruzados.
